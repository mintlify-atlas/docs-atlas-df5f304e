## Hardware generations

nRFBox has gone through three hardware revisions:

| Version | Platform | Status |
|---|---|---|
| V1 | Arduino | No longer maintained |
| V2 | ESP32 | Firmware actively updated; pre-compiled binaries available |
| V3 | ESP32 | Latest hardware design, referenced in README |

V1 was the original Arduino-based implementation. V2 migrated to the ESP32, gaining WiFi, BLE, and expanded memory. V3 is the latest PCB revision, also ESP32-based, incorporating hardware refinements over V2.

### V2.7.2 — latest recommended release

`nRFBox_V2-7-2.ino.node32s.bin`

The most recent pre-compiled release. Recommended for all users flashing directly to hardware. Use the flash tool and partition binary provided in the `Flash File/` directory.

### V2.5.0

`nRFBox_V2-5-0.ino.node32s.bin`

An intermediate release in the V2 firmware series, reflecting continued development and refinement between V2.2.1 and V2.7.2.

### V2.2.1

`nRFBox_V2-2-1.ino.node32s.bin`

An incremental release in the V2 series.

### V2.1.1

`nRFBox_V2-1-1.ino.node32s.bin`

An early update to the initial V2 ESP32 release.

### V2.0.0 — initial ESP32 release

`nRFBox_V2-0-0.ino.node32s.bin`

The first firmware release for the ESP32-based nRFBox V2 hardware. Introduced the core feature set: Scanner, Analyzer, Jammer, BLE Jammer, BLE Spoofer, and Sour Apple.

## Flashing

Pre-compiled binaries can be flashed using the Espressif Flash Download Tool (`flash_download_tool_3.9.5.zip`) included in the `Flash File/` directory, along with the required `nRFBOX.partitions.bin` partition table and `platform.txt` configuration file. See the [project wiki](https://github.com/cifertech/nRFBox/wiki) for flashing instructions.

OTA (over-the-air) firmware updates from an SD card are supported in V2 firmware via the Settings menu. Place `firmware.bin` on the SD card root and select "Update Firmware" from the on-device settings.
