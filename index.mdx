## Welcome to nRFBox

nRFBox is an open-source hardware toolkit built around the ESP32 and three NRF24L01 radio modules. It lets you explore, analyze, and interact with wireless protocols across the 2.4GHz spectrum — including BLE, Wi-Fi, Zigbee, and more — all from a compact handheld device with an OLED display.

<Warning>
  nRFBox is intended for security research, authorized penetration testing, and educational use only. Using jamming or spoofing features against networks or devices you do not own is illegal in most jurisdictions. Always obtain proper authorization before testing.
</Warning>

## Get started in three steps

<Steps>
  <Step title="Assemble the hardware">
    Connect the ESP32 Wroom32U, three NRF24L01 modules, SSD1306 OLED, NeoPixel LED, and navigation buttons. See the [Hardware Setup](/hardware-setup) guide for the full wiring diagram.
  </Step>
  <Step title="Flash the firmware">
    Use the Arduino IDE with ESP32 board support, or flash a pre-compiled `.bin` file directly using `esptool.py`. Full instructions are in the [Flashing Firmware](/flashing-firmware) guide.
  </Step>
  <Step title="Select a mode and start exploring">
    Use the five navigation buttons to browse the icon grid menu and select one of the 10 operational modes. Press **Select** to enter a mode, and press **Select** again to return to the main menu.
  </Step>
</Steps>
